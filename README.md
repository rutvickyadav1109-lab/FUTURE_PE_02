# FUTURE_PE_02 вЂ” Prompt Engineering Internship Task

## Project Overview

This project is part of my Prompt Engineering Internship task.  
The goal was to write effective prompts for real-world copywriting tasks for a local business client, and then compare the outputs generated by 3 different AI models вЂ” Claude, ChatGPT, and Gemini.

**Client:** Maya Garments, Boisar, Maharashtra (Men + Boys clothing store)  
**Brands Available:** Zara, H&M, Ralph Lauren, US Polo, Snitch, Urban Monkey, Van Heusen, Rare Rabbit, Gant, Powerlook + more  
**Target Audience:** Young men, boys, and families in and around Boisar

## Key Learnings

- Writing clear, specific prompts gives better and more consistent outputs
- Tone instructions (Hinglish, warm, energetic) directly affect output quality
- Each AI model has a different style вЂ” Claude felt more natural and local, ChatGPT was more structured, Gemini was more formal
- The same prompt framework can be adapted across different business types and industries
