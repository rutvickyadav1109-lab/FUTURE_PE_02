# FUTURE_PE_02 — Prompt Engineering Internship Task

## Project Overview

This project is part of my Prompt Engineering Internship task.  
The goal was to write effective prompts for real-world copywriting tasks for a local business client, and then compare the outputs generated by 3 different AI models — Claude, ChatGPT, and Gemini.

**Client:** Maya Garments, Boisar, Maharashtra (Men + Boys clothing store)  
**Brands Available:** Zara, H&M, Ralph Lauren, US Polo, Snitch, Urban Monkey, Van Heusen, Rare Rabbit, Gant, Powerlook + more  
**Target Audience:** Young men, boys, and families in and around Boisar

---

## 5 Prompt Tasks

| # | Task | File |
|---|------|------|
| 1 | UGC Hooks for Instagram / Reels / YouTube Shorts | `prompts/01_hooks_prompt.txt` |
| 2 | UGC Ad Scripts (4 characters, 4 platforms) | `prompts/02_ad_scripts_prompt.txt` |
| 3 | CTA Sections (5 different types) | `prompts/03_cta_prompt.txt` |
| 4 | Platform-Specific Captions (Instagram, Facebook, YouTube) | `prompts/04_captions_prompt.txt` |
| 5 | Tone Adaptation Framework (3 business types) | `prompts/05_tone_adaptation_prompt.txt` |

## Key Learnings

- Writing clear, specific prompts gives better and more consistent outputs
- Tone instructions (Hinglish, warm, energetic) directly affect output quality
- Each AI model has a different style — Claude felt more natural and local, ChatGPT was more structured, Gemini was more formal
- The same prompt framework can be adapted across different business types and industries
